# Farmer Class

A small, standalone Vintage Story mod that adds a single farmer character class.

![farmer](farmer.png)

## Build

```sh
./build.sh
VS_DIR=/path/to/vs ./build.sh
```

Produces `dist/farmerclass_<version>.zip`
